# SR06
This App does Voice Recognizition and Recognize what you said. Convert that to Text and matches with given algo.
Search feature is incomplete, ideally you can search by saying : "search "keyword""
Currently on searching it returns result in textual form. One need to parse the result to get desired output.
App does not end until you say exit. Yes you have to say!!
You can also pause by just saying "pause"

It's interactive and fun to talk with.
